# Number Guessing Game A simple Python game where the computer selects a random number
   and the user tries to guess it.
   
## Features 
  - Random number generation
  - Unlimited attempts
  - Hint system (Too High / Too Low)
  - Attempt counter

    
## Technologies Used
 - Python
 - VS Code

## How to Run 
```bash python main.py'''
